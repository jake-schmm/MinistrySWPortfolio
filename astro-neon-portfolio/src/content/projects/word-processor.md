## Overview
WordProcessor is a document-writing application that lets users create rich-text documents, save their work, and share documents with others based on visibility settings.

## Core Features

- Create and edit rich-text documents
- Save documents for later access
- Share documents with other users
- Set document visibility levels
- View documents based on access permissions

## Visibility Levels

### Public

Anyone with access to the site can view the document.

### Self-Only

Only the document owner can view and edit the document.

### Friends-Only

Only approved friends can view the document.

## Project Goal

The goal of this project is to create a clean, focused writing experience while also supporting controlled sharing. Instead of every document being either fully private or fully public, users can choose the level of visibility that fits the purpose of the document.
It was also designed to practice layered architecture for the first time.

## Example Use Cases

- A user writes a private journal entry and marks it as **Self-Only**
- A user creates a tutorial and marks it as **Public**
- A user shares a draft with friends by marking it as **Friends-Only**
- A user edits and saves a document over time
