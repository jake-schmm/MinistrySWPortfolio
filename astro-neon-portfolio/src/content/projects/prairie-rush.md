## Overview

Prairie Rush was a single-player game built in Unity for a User Interaction Engineering course. The project emphasized intuitive gameplay, responsive controls, and user-friendly UI design while maintaining a fun retro atmosphere.

## How to Access

Please contact me directly for the project's code. Otherwise, you can use the "Download" link above, which will redirect you to my itch.io profile where you can download the game for Windows. It will say untrusted publisher at the moment, as I don't have formal credentials or enough people who have played the game yet.

## My Contributions

Although I was not the primary gameplay developer, I designed and programmed a significant portion of the project, such as: 
- Jumping
- Prairie-inspired textures
- Health bar and obstacle-hitting logic
- Integrating music
- Game Over Screen
- Accessibility Settings (such as turning "health bar flash" on/off and using less intense textures) 

I also integrated animations from an Asset store and refactored UI logic to make it more maintainable.

## User Interaction Engineering Goals

The primary goal of the course project was to design a user-friendly experience. This included:

- Clear visual feedback
- Easy-to-use controls
- Understandable menus and HUD elements
- Smooth gameplay interaction
